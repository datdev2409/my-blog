## Exam Overview

CKS: performance based exam test knowledge about Kubernetes and cloud security

### Requirement:

- No longer need the active CKA certificate, you just need to achieve the CKA at any time (Updates from Sep 2024)
- 67% score to pass the exam

### Domain & Weighting

| Domain                                   | Weight |
| ---------------------------------------- | ------ |
| Cluster Setup                            | 10%    |
| Cluster Hardening                        | 15%    |
| System Hardening                         | 15%    |
| Minimize Microservice Vulnerabilities    | 20%    |
| Supply Chain Security                    | 20%    |
| Monitoring, Logging and Runtime Security | 20%    |
